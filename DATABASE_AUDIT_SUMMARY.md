### Data Migration Tasks

1. **Sticker Number Backfill**

   ```sql
   -- For each collection, assign sequential numbers
   UPDATE stickers s
   SET sticker_number = subq.rn
   FROM (
     SELECT id, ROW_NUMBER() OVER (
       PARTITION BY collection_id
       ORDER BY id
     ) as rn
     FROM stickers
   ) subq
   WHERE s.id = subq.id
   AND s.sticker_number IS NULL;
   ```
